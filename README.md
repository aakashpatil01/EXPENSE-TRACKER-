# EXPENSE-TRACKER-
A handy personal finance tool built in Python that  helps you record and analyse your daily spending.


## Project Overview
Keeping track of expenses is something everyone 
struggles with. I built this tool to make it easier
to record where money is being spent and understand
spending patterns through a simple category report.
This was the most practical project I built during
my internship.

## Features at a Glance
- Log a new expense with date, category and amount
- View all recorded expenses in a table
- Remove an expense entry you added by mistake
- Check the total amount spent so far
- Get a full breakdown of spending by category

## Technologies Used
- Python 3
- No external libraries needed

## Getting Started
Step 1 — Make sure Python 3 is installed
Step 2 — Download or clone this repository
Step 3 — Open terminal and run:
         python expense_tracker.py
